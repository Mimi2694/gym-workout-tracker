Project Overview:

The FitTrack Pro – Gym & Workout Tracker System is designed to help users track their workouts, exercises, and fitness progress efficiently.

The system allows users to:

Register and login securely

Create and manage workout plans

Record exercises (sets, reps, weight)

Track body weight and performance progress

Monitor improvement through structured tracking

The system aims to improve fitness organization, consistency, and performance analysis by providing a structured and user-friendly tracking platform.
